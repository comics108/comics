## ОТЛИЧИЯ ОТ COMICS EDITOR

1. **Модели данных:** Создаются заново (PuzzlePiece, PuzzleGrid), а не переписываются с C#
2. **Меньше файлов для переписывания:** Нет анимаций, звуков - только кусочки пазла
3. **Общий C# handler:** Используется тот же native/Comics.Editor/, что и comics
4. **Разные FFI функции:** puzzle_* функции вместо editor_* функций
5. **Генерация из изображения:** Нужен image processor для нарезки изображения на кусочки
